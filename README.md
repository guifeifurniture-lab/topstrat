# 谋士天团官网

> 8 位 AI 谋士。一家公司。

## 🚀 快速开始

```bash
# 安装依赖
npm install

# 开发模式
npm run dev

# 构建生产版本
npm run build

# 部署到 Vercel
npm run deploy
```

## 📁 项目结构

```
website/
├── app/                    # Next.js 14 App Router
│   ├── layout.tsx          # 根布局
│   ├── page.tsx            # 主页
│   ├── office/
│   │   └── page.tsx        # 谋士堂
│   ├── radar/
│   │   └── page.tsx        # 需求雷达
│   ├── vault/
│   │   └── page.tsx        # 锦囊库
│   └── insights/
│       └── page.tsx        # 场记
├── components/             # React 组件
│   ├── Nav.tsx             # 导航栏
│   ├── Footer.tsx          # 页脚
│   ├── Hero.tsx            # Hero 区域
│   ├── AgentCard.tsx       # 谋士卡片
│   └── PricingCard.tsx     # 定价卡片
├── lib/                    # 工具函数
│   └── utils.ts
├── public/                 # 静态资源
│   └── images/
├── styles/                 # 全局样式
│   └── globals.css
├── next.config.js          # Next.js 配置
├── tailwind.config.js      # Tailwind 配置
├── tsconfig.json           # TypeScript 配置
└── package.json
```

## 🛠️ 技术栈

- **框架:** Next.js 14 (App Router)
- **语言:** TypeScript
- **样式:** Tailwind CSS
- **部署:** Vercel
- **分析:** Vercel Analytics
