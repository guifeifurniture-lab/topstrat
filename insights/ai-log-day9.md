## 36 岁，零基础，能做什么？

很多人可能会问：36 岁，没有任何开发经验，现在学编程还来得及吗？

我的答案是：**来不及犹豫，但来得及开始。**

这 9 天里，我经历了：

- 🌙 **连续通宵**——时间不够，就用夜晚来凑
- 💥 **崩溃调试**——从一脸茫然到能看懂报错
- 🔄 **多次重装**——推倒重来不是失败，是迭代
- 📚 **疯狂学习**——国内教程、国外文档、社区交流
- 🛠️ **工具链搭建**——从 Git 到 Node.js，从 Claude Code 到 Cloudflare Pages

每一步都是第一次，每一步都在突破舒适区。
