# 支付二维码配置说明

## 目录结构
```
website/
└── assets/
    ├── alipay-qr.png        # 支付宝收款二维码（需要添加）
    ├── wechat-pay-qr.png    # 微信支付收款二维码（需要添加）
    └── payment-qr-instructions.md  # 本说明文件
```

## 支付宝二维码配置

### 获取步骤
1. 打开支付宝 App
2. 点击「我的」→「收付款」→「二维码收款」
3. 点击「保存收款码」或截图保存
4. 将图片重命名为 `alipay-qr.png`
5. 放置到 `website/assets/` 目录

### 收款账户信息
- **账户名称：** GuiFei Furn 经营账户
- **支持支付方式：** 信用卡、花呗、支付宝余额、借记卡

## 微信支付二维码配置

### 获取步骤
1. 打开微信 App
2. 点击「我」→「服务」→「收付款」→「二维码收款」
3. 点击「保存收款码」或截图保存
4. 将图片重命名为 `wechat-pay-qr.png`
5. 放置到 `website/assets/` 目录

### 收款账户信息
- **账户名称：** Michael 经营账户
- **支持支付方式：** 微信零钱、信用卡、借记卡

## 二维码图片要求

- **格式：** PNG（推荐）或 JPG
- **尺寸：** 建议 256x256 像素或更大
- **质量：** 确保清晰可扫描，边缘锐利
- **背景：** 白色背景最佳

## 测试验证

放置二维码图片后，访问 `vault.html` 页面验证：
1. 打开 `http://localhost/vault.html`（或你的服务器地址）
2. 点击「微信支付」或「支付宝」选项卡
3. 确认二维码正常显示
4. 使用手机扫描测试（应显示正确的收款账户信息）

## 支付流程

### 用户支付流程
1. 用户在 vault.html 选择产品（入门锦囊 ¥299 / 专业锦囊 ¥799）
2. 选择支付方式（PayPal / 微信支付 / 支付宝）
3. 扫码完成支付
4. 点击「我已支付」按钮
5. 系统记录支付信息

### 管理员确认流程
1. 管理员收到支付确认通知（邮件 / 系统通知）
2. 核实收款账户到账情况
3. 手动添加用户为 GitHub Collaborator
4. 发送确认邮件给用户
5. 更新支付状态为已完成
