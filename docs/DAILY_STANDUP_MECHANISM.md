## 🌅 晨会（05:00）

### 议程（15 分钟）

1. **昨日回顾**（3 分钟）
   - 昨日完成的任务
   - 未完成的任务及原因

2. **今日计划**（10 分钟）
   - 各 Agent 今日目标
   - 需要协作的事项
   - 潜在风险预警

3. **任务分配**（2 分钟）
   - 确认各 Agent 任务
   - 设定优先级
