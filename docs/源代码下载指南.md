## 🚀 快速启动

### 官网源码（Next.js）

```bash
# 安装依赖
cd topstrat
npm install

# 开发模式
npm run dev

# 构建
npm run build

# 生产部署
npm run start
```

**访问：** http://localhost:3000

---

### CrewAI 框架（Python）

```bash
# 创建虚拟环境
cd crewai
python -m venv venv
venv\Scripts\activate  # Windows
source venv/bin/activate  # Mac/Linux

# 安装依赖
pip install -r requirements.txt

# 配置环境变量
cp .env.example .env
# 编辑.env 文件，填写 API 密钥

# 运行 CrewAI
python crews/website_crew.py
```

## 🆘 故障排除

### 问题：Git Clone 失败

**解决：**
```bash
# 检查网络连接
ping github.com

# 使用 HTTPS 代替 SSH
git clone https://github.com/guifeifurniture-lab/topstrat.git

# 或配置 Git 代理
git config --global http.proxy http://proxy.example.com:8080
```

### 问题：依赖安装失败

**解决：**
```bash
# 升级 pip
pip install --upgrade pip

# 清理缓存后重新安装
pip cache purge
pip install -r requirements.txt --force-reinstall
```

### 问题：端口被占用

**解决：**
```bash
# 查找占用端口的进程
netstat -ano | findstr :3000

# 终止进程
taskkill /PID <进程 ID> /F

# 或使用其他端口
npm run dev -- -p 3001
```
