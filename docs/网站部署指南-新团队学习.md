## 🚀 部署步骤

### 阶段 1：获取源代码

#### 方法 1：Git Clone（推荐）

```bash
# 克隆官网源码
git clone https://github.com/guifeifurniture-lab/topstrat.git
cd topstrat

# 克隆 CrewAI 框架（可选）
git clone https://github.com/guifeifurniture-lab/moushi-crewai.git
cd moushi-crewai
```

#### 方法 2：GitHub 网页下载

1. 访问 https://github.com/guifeifurniture-lab/topstrat
2. 点击绿色 **"Code"** 按钮
3. 选择 **"Download ZIP"**
4. 解压到本地目录

---

### 阶段 2：本地开发环境搭建

#### 步骤 1：安装依赖

```bash
cd topstrat
npm install
```

**预计时间：** 2-5 分钟  
**输出：** `node_modules/` 目录

#### 步骤 2：启动开发服务器

```bash
npm run dev
```

**访问：** http://localhost:3000  
**功能：** 实时热更新，修改代码自动刷新

#### 步骤 3：验证功能

- [ ] 首页正常显示
- [ ] 定价页（/vault）正常显示
- [ ] 博客页（/insights）正常显示
- [ ] 响应式布局正常（移动端/桌面端）

---

### 阶段 3：构建生产版本

#### 步骤 1：执行构建

```bash
npm run build
```

**预计时间：** 1-3 分钟  
**输出：** `.next/` 目录（生产优化版本）

#### 步骤 2：本地测试生产版本

```bash
npm run start
```

**访问：** http://localhost:3000  
**验证：** 确保生产版本功能正常

---

### 阶段 4：部署到 Cloudflare Pages

#### 步骤 1：登录 Cloudflare

1. 访问 https://dash.cloudflare.com
2. 登录/注册账号

#### 步骤 2：创建 Pages 项目

1. 点击左侧 **"Workers & Pages"**
2. 点击 **"Create application"**
3. 选择 **"Pages"** 标签
4. 点击 **"Connect to Git"**

#### 步骤 3：连接 GitHub 仓库

1. 选择 **"GitHub"**
2. 授权 Cloudflare 访问 GitHub
3. 选择仓库：`guifeifurniture-lab/topstrat`
4. 点击 **"Begin setup"**

#### 步骤 4：配置构建设置

| 配置项 | 值 | 说明 |
|--------|-----|------|
| **Framework preset** | Next.js | 自动识别 |
| **Build command** | `npm run build` | 生产构建 |
| **Build output directory** | `.next` | 输出目录 |
| **Root directory** | （留空） | 默认根目录 |

#### 步骤 5：环境变量（可选）

如需 AI 功能，添加环境变量：

| 变量名 | 值 | 说明 |
|--------|-----|------|
| `OPENAI_API_KEY` | `sk-...` | OpenAI API 密钥 |
| `BRAVE_SEARCH_API_KEY` | `...` | Brave Search API 密钥 |

#### 步骤 6：保存并部署

1. 点击 **"Save and Deploy"**
2. 等待构建完成（约 2-5 分钟）
3. 获取部署域名：`https://topstrat-xxxx.pages.dev`

---

### 阶段 5：自定义域名（可选）

#### 步骤 1：添加域名

1. 进入 Pages 项目设置
2. 点击 **"Custom domains"**
3. 点击 **"Set up a custom domain"**
4. 输入域名：`yourdomain.com`

#### 步骤 2：配置 DNS

1. Cloudflare 自动生成 DNS 记录
2. 等待 DNS 传播（约 5-30 分钟）
3. SSL 证书自动签发

#### 步骤 3：验证

访问 `https://yourdomain.com` 确认部署成功

## 🛠️ 故障排除

### 问题 1：构建失败

**错误信息：** `npm ERR! code ELIFECYCLE`

**解决方案：**
```bash
# 清理缓存
npm cache clean --force

# 删除 node_modules
rm -rf node_modules package-lock.json

# 重新安装
npm install

# 重新构建
npm run build
```

---

### 问题 2：Cloudflare 部署失败

**错误信息：** `Error: Build failed`

**解决方案：**
1. 检查 `.gitignore` 文件
2. 确保 `node_modules` 未被提交
3. 查看构建日志定位错误
4. 本地复现构建错误

---

### 问题 3：页面 404

**解决方案：**
1. 检查路由配置（`next.config.ts`）
2. 确认页面文件存在（`src/app/xxx/page.tsx`）
3. 清除浏览器缓存
4. 检查 Cloudflare 缓存设置

---

### 问题 4：样式丢失

**解决方案：**
1. 检查 Tailwind CSS 配置
2. 确认 `globals.css` 已导入
3. 检查构建输出目录配置
4. 清除 CDN 缓存
