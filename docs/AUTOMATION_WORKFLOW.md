## 🚨 异常处理

### 场景 1：Agent 无响应

**检测：** 15 分钟心跳未回复  
**处理：**
1. 诸葛亮记录异常
2. 重试发送任务（最多 3 次）
3. 仍无响应 → 报告老庄
4. 临时调整任务分配

### 场景 2：部署失败

**检测：** 商鞅监控告警  
**处理：**
1. 回滚到上一个稳定版本
2. 分析失败原因
3. 修复后重新部署
4. 记录故障报告

### 场景 3：Cron 任务停止

**检测：** Office 数据超过 10 分钟未更新  
**处理：**
1. 商鞅重启 Cron 服务
2. 检查任务日志
3. 修复配置问题
4. 验证恢复情况

## 🎯 启动命令

### 启动全员工作

```bash
# 1. 启用 Cron 服务
openclaw cron enable

# 2. 注册定时任务
openclaw cron register --file cron/cron-config.json

# 3. 启动各 Agent
openclaw run architect  # 管仲
openclaw run delivery   # 孙膑
openclaw run ops        # 商鞅
openclaw run qa         # 萧何
openclaw run content    # 张良
openclaw run sales      # 范蠡

# 4. 派发初始任务
openclaw send architect "审查 website/ 代码质量"
openclaw send delivery "优化 Stage 页面动画"
openclaw send ops "配置 Cron 监控"
openclaw send qa "测试所有页面功能"
openclaw send content "分析 voxyz.space 最新变化"
openclaw send sales "统计 Vault 页面访问数据"
```
