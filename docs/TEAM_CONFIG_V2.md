### 启动全部 Agent

```bash

$agents = @("zhugeliang","guanzhong","sunbin","guojia","mozi","shangyang","xiaohe","zhangliang","fanli","hanxin","chenping")

    Start-Process "openclaw" -ArgumentList "run $agent"
}
```
