## 🎯 分配原则

1. **角色匹配:** 任务与谋士角色专长匹配
   - CEO → 汇报类任务
   - CTO → 技术学习类
   - DevOps → 系统监控类
   - CFO → 财务投资类

2. **负载均衡:** 人均 2 个任务，避免过度集中

3. **时间分散:** 不同频率任务组合，避免同时触发

4. **技能互补:** 相关任务分配给同一谋士，形成知识积累
